# MineSweeper TAS для ClassiCube

Автоматический решатель головоломки Minesweeper, встроенный в клиент ClassiCube.

## Описание

**MineSweeper TAS** (Tool-Assisted Speedrun) - это система автоматического решения головоломки Minesweeper на сервере ClassiCube. Система предназначена для мини-игры на платформе размером 9x1x9 блоков.

### Как это работает

- **Размер платформы**: 9x1x9 блоков
- **Активация**: Нажмите **Numpad 5** для включения/отключения TAS
- **Автоматизм**: TAS самостоятельно сканирует поле, применяет логику Minesweeper и кликает по безопасным клеткам

## Блоки Minesweeper

Система использует следующие ID блоков для представления состояния головоломки:

| Тип клетки | ID блока | Описание |
|-----------|----------|---------|
| Нераскрытая | 43 | Закрытая клетка (не открыта) |
| Пусто | 1 | Раскрыта, соседей с минами нет |
| 1 сосед | 28 | Рядом 1 мина |
| 2 соседа | 25 | Рядом 2 мины |
| 3 соседа | 21 | Рядом 3 мины |
| 4 соседей | 29 | Рядом 4 мины |
| 5 соседей | 22 | Рядом 5 мин |
| 6 соседей | 27 | Рядом 6 мин |
| 7 соседей | 34 | Рядом 7 мин |
| 8 соседей | 35 | Рядом 8 мин |

## Как использовать

### Установка

1. Убедитесь, что файлы добавлены в проект:
   - `src/MineSweeper_TAS.h` - заголовочный файл
   - `src/MineSweeper_TAS.c` - реализация
   - `src/InputHandler.c` - обновлен для интеграции

2. Пересоберите проект ClassiCube

### Использование в игре

1. Встаньте в центр платформы 9x1x9 (прямо на неё, а не внутри)
2. Нажмите **Numpad 5** для активации TAS
3. В чате появится сообщение `&eMineSweeper TAS &aACTIVATED`
4. TAS автоматически:
   - Сканирует поле
   - Применяет логику Minesweeper
   - Кликает по безопасным клеткам
   - Решает головоломку
5. Нажмите **Numpad 5** ещё раз для деактивации

## Алгоритм решения

TAS использует три стратегии решения в порядке приоритета:

### Стратегия 1: Раскрытие после флагов
Если количество отмеченных мин (флагов) равно числу на раскрытой клетке, все остальные нераскрытые соседи безопасны → раскрываем их.

### Стратегия 2: Отметка мин
Если количество нераскрытых клеток + отмеченных мин равно числу на раскрытой клетке, все оставшиеся нераскрытые - это мины → отмечаем их.

### Стратегия 3: Поиск безопасных клеток
Ищем нераскрытые клетки, которые безопасны благодаря логике Minesweeper (соседняя клетка уже имеет все мины отмечены).

## Сообщения

- `MineSweeper TAS initialized. Press Numpad 5 to activate.` - TAS инициализирован
- `&eMineSweeper TAS &aACTIVATED` - TAS активирована
- `&eMineSweeper TAS &cDEACTIVATED` - TAS отключена
- `&aMineSweeper puzzle solved!` - Головоломка решена

## Структура кода

### MineSweeper_TAS.h
- `struct MineSweeper_Cell` - информация о клетке (позиция, блок, значение, флаги)
- `struct MineSweeper_TAS` - главная структура системы
- Функции API для управления и обновления

### MineSweeper_TAS.c
- `MineSweeper_Init()` - инициализация системы
- `MineSweeper_Activate()` - активация TAS
- `MineSweeper_Deactivate()` - деактивация TAS
- `MineSweeper_ScanBoard()` - сканирование поля из блоков
- `MineSweeper_Solve()` - применение алгоритма решения
- `MineSweeper_ClickCell()` - выполнение клика по клетке
- `MineSweeper_Update()` - основной цикл обновления

### InputHandler.c
- Интеграция обработки Numpad 5
- Вызов `MineSweeper_Update()` в `InputHandler_Tick()`
- Инициализация в `OnInit()`

## Технические подробности

### Система координат
- Платформа начинается на **center ± 4** по X и Z
- Y координата фиксирована (уровень платформы)
- Доска представлена как массив 9×9 структур `MineSweeper_Cell`

### Тайминг
- Задержка между сканированиями: 100 мс
- Задержка между кликами: 150 мс
- Задержка после завершения: 5000 мс

### Ограничения
- Работает только на платформе 9x1x9
- Требует корректной разметки блоков (ID должны соответствовать таблице выше)
- Работает в один поток с основным игровым циклом
